# @youngjuning/create-commitlint

## Usage directly

```sh
$ npx @youngjuning/create-commitlint
# or
$ npm init @youngjuning/commitlint
# or
$ yarn create @youngjuning/commitlint
```

## Install before Usage

```sh
# install
$ npm install @youngjuning/create-commitlint -g
# usage
$ create-commitlint
```
