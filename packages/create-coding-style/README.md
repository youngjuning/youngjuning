# @zisheng/create-coding-style

## Usage directly

```sh
$ npx @zisheng/create-coding-style
# or
$ npm init @zisheng/create-coding-style
# or
$ yarn create @zisheng/coding-style
```

## Install before Usage

```sh
# install
$ npm install @zisheng/create-coding-style -g
# usage
$ create-coding-style
```
